## 设计原则

所有设计都遵循苹果官网的设计语言：

1. **简洁干净**
   - 大量留白
   - 极简设计元素
   - 去除不必要的装饰

2. **大量留白**
   - 容器内边距充足
   - 元素间距合理
   - 视觉呼吸感

3. **柔和阴影**
   - 使用极低透明度的黑色阴影
   - 多层阴影增加层次感
   - 悬浮时阴影增强

4. **大圆角**
   - 卡片圆角24px
   - 按钮完全圆形
   - 输入框圆角12px

5. **优雅动画**
   - 缓动函数使用cubic-bezier(0.4, 0, 0.2, 1)
   - 动画时长合理（0.2s-0.6s）
   - 流畅的过渡效果

6. **精致排版**
   - 使用负字间距（letter-spacing: -0.02em）
   - 合理的行高（line-height: 1.6）
   - 响应式字体大小

## 颜色系统

### 主色调
- 背景: #ffffff (纯白)
- 文字: #1d1d1f (深灰)
- 边框: rgba(0, 0, 0, 0.06) (极细透明)
- 主色: #0071e3 (苹果蓝)

### 阴影系统
- 默认: 0 2px 12px rgba(0, 0, 0, 0.04)
- 悬浮: 0 8px 32px rgba(0, 0, 0, 0.08)
- 强调: 0 12px 48px rgba(0, 0, 0, 0.1)

### 圆角系统
- 卡片: 24px
- 按钮: 9999px (完全圆形)
- 输入框: 12px
- 小元素: 8px

## 动画系统

### 缓动函数
- 所有动画使用: cubic-bezier(0.4, 0, 0.2, 1)

### 动画类型
- 淡入: apple-fade-in (0.5s)
- 滑动: apple-slide-up (0.6s)
- 按缩放: scale(0.97) (0.1s)
